🚀 GitHub Pages 배포 가이드

                    
                    

                        
1️⃣ GitHub 레포지토리 생성

                        
    
                            
  1. GitHub.com 접속 후 로그인
    2. 
                            
  2. "New repository" 클릭
    3. 
                            
  3. Repository name: building-tool (원하는 이름)
    4. 
                            
  4. "Public" 선택
    5. 
                            
  5. "Create repository" 클릭
    6. 
                        

                    

                    
                    

                        
2️⃣ 파일 업로드

                        
    
                            
  1. 레포지토리 페이지에서 "uploading an existing file" 클릭
    2. 
                            
  2. 이 HTML 파일을 index.html로 저장
    3. 
                            
  3. 파일을 드래그 앤 드롭
    4. 
                            
  4. "Commit changes" 클릭
    5. 
                        

                        
                        

📁 building-tool/
  └── index.html
  └── buildings.json (선택)
                        

                    

                    
                    

                        
3️⃣ GitHub Pages 활성화

                        
    
                            
  1. Settings 탭 클릭
    2. 
                            
  2. 왼쪽 메뉴에서 "Pages" 클릭
    3. 
                            
  3. Source: "Deploy from a branch" 선택
    4. 
                            
  4. Branch: "main" / "root" 선택
    5. 
                            
  5. "Save" 클릭
    6. 
                        

                        
                        

                            💡 1-2분 후 아래 주소로 접속 가능:
                            

                                https://[github-username].github.io/building-tool/
                            

                        

                    

                    
                    

                        
4️⃣ 카카오맵 API 키 설정 (선택)

                        
    
                            
  1. Kakao Developers 접속
    2. 
                            
  2. 앱 만들기 → JavaScript 키 복사
    3. 
                            
  3. 앱 설정 → 플랫폼 → Web → 사이트 도메인 추가
    4. 
                            
  4. https://[username].github.io 입력
    5. 
                        

                        
                        

                            ⚠️ 카카오맵 API 없이도 수동 좌표 입력으로 사용 가능합니다!
                        

                    

                    
                    

                        
5️⃣ 로컬에서 테스트하기

                        
                        
VS Code 사용시:

                        
    
                            
  1. VS Code에서 폴더 열기
    2. 
                            
  2. Live Server 확장 설치
    3. 
                            
  3. index.html 우클릭 → "Open with Live Server"
    4. 
                        

                        
                        
Python 사용시:

                        

# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# 브라우저에서 http://localhost:8000 접속
                        

                        
                        
Node.js 사용시:

                        

# http-server 설치
npm install -g http-server

# 실행
http-server

# 브라우저에서 http://localhost:8080 접속
